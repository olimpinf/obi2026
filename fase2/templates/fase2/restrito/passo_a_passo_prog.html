Passo a passo: aplicação da Prova Fase 2 - Modalidade Programação

Antes da prova

Preparação dos laboratórios

  1. Para desenvolver suas soluções, cada aluno deve ter à sua disposição um computador com editores de texto e pelo menos um compilador entre C, C++, Java ou o interpretador Python. Normalmente os alunos usam o que já está instalado nos laboratórios da escola. Se a escola não tem compiladores ou ambientes de desenvolvimento já instalados, veja em {% url 'aplicativos' %} alguns aplicativos algumas sugestões de aplicativos que podem ser instalados.
  2. Os alunos acessam o Ambiente de Prova usando um navegador internet (como Firefox, Chrome, Safari, etc.).
  3. O aluno não deve acessar a Internet a não ser através do Ambiente de Prova da {{ 'OBI'|obi_date_year }}. Se possível, o ideal é permitir o acesso apenas aos endereços do Ambiente de Prova da OBI (veja abaixo). Se isso não é possível, os professores devem supervisionar os alunos de forma a que não acessem outros endereços da Internet.
  4. Os alunos devem fazer as provas individualmente, sem consulta a qualquer material impresso (como livros ou apostilas) ou digital (como programas feitos anteriormente).

Caso Especial: Provas no ambiente Saci

Alguns (poucos) alunos estudaram programação pelo ambiente Saci, da OBI, que tem dois cursos disponívels, Javascript e Blockly/Python Para esses alunos, preparamos uma versão especial do Editor Saci para as provas da OBI, para que os alunos tenham o mesmo ambiente e para que possam desenvolver e testar suas soluções em Javascript ou Python.

Se algum aluno de sua escola solicitar acesso ao Editor Saci, siga os passos:

  • Faça o download do Editor Saci no link abaixo (arquivo .zip)
  • Copie o arquivo .zip para o computador onde o aluno vai fazer a prova (por exemplo, saci.zip).
  • Descompacte o arquivo. Vai ser criada uma pasta (de nome saci)
  • Dentro da pasta criada, clique duas vezes no arquivo de nome "index.html"; o navegador padrão (Firefox, Chrome, Opera...) do sistema deve abrir e mostrar a prova.
  • Se não funcionar, abra uma janela de um navegador (Firefox, Chrome...) e abra o arquivo "index.html" pelo navegador. O ambiente de desenvolvimento deve aparecer como uma página do navegador.
    • Editor Saci Javascript
    • Editor Saci Blockly/Python (infelizmente não é possível disponibilizar o editor como um arquivo. Instrua os seus competidores a utilizarem o Editor Blockly/Python disponível no site da OBI.

No dia da prova

  1. Baixe e imprima as senhas de acesso ao Ambiente de Prova.
    1. Na página de coordenação, Seção "Fase 2", linha "Modalidade Programação", clique em "Senhas Prova". Um arquivo PDF no formato de "cartões" é gerado, cada cartão com as informações de um competidor (nome, usuário, senha).
  2. Imprima as Listas de Presença
    1. As listas de presença não são obrigatórias, mas é recomendável que os alunos assinem. Guarde as listas assinadas para comprovar a participação dos alunos em caso de problemas com a prova ou a correção.
    2. Para imprimir as listas de presença, na página de coordenação, Seção "Fase 2", linha "Modalidade Programação", clique em "Aplicação da Prova", e procure a seção Listas de Presença.
  3. Distribua as senhas para os alunos.
    • Os cartões estão no formato de etiquetas auto-colantes, mas podem ser recortados manualmente para serem entregues para os alunos.
  4. O nome de usuário é o número de inscrição (completo, por exemplo 01234-B).
  5. Informe os endereços do Ambiente de Prova para os alunos (veja os endereços abaixo).
  6. Supervisione os alunos, para garantir que façam a prova individualmente e sem uso de qualquer material externo (livros, internet, anotações, arquivos de problemas já feitos, etc)
  7. O tempo de prova, controlado pelo Ambiente de Prova, é duas horas para os níveis Júnior e 1, e três horas para os níveis 2 e Sênior.

Endereços do Ambiente de prova

  • São utilizados quatro endereços diferentes para o Ambiente de Prova, um para cada nível da modalidade Programação:
    • Nível Júnior: https://pj.provas.ic.unicamp.br
    • Nível 1: https://p1.provas.ic.unicamp.br
    • Nível 2: https://p2.provas.ic.unicamp.br
    • Nível Sênior: https://ps.provas.ic.unicamp.br
  • O ambiente de prova estará aberto das 08:00 às 20:00 (horário estendido a pedido das escolas). Note que o ambiente FECHA às 20:00, para todos os níveis, não sendo possível submeter soluções após esse horário.

Resultados da Prova

  1. Após o período de aplicação da prova, competidores e coordenadores locais poderão ver a correção das soluções submetidas.
  2. Os resultados são provisórios, pois pode haver re-julgamento das soluções em caso de necessidade.
  3. Os resultados finais serão divulgados de acordo com o calendário.
