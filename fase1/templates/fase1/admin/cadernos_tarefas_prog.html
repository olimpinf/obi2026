Cadernos de Tarefas - Fase 1 - Modalidade Programação

Para cada competidor, é necessário imprimir uma cópia do Caderno de Tarefas. Ao final da prova, o Coordenador Local deve recolher as soluções dos alunos, preparar um arquivo com as soluções e enviar para correção, utilizando a página Modalidade Programação - Correção (ainda não disponível, será disponibilizada logo após a prova).

Veja a seguir um passo-a-passo detalhado sobre o procedimento de aplicação e correção das provas da Modalidade Programação.

Passo-a-passo

  • Pegue o arquivo do Caderno de Tarefas (prova) nos links abaixo.
  • Faça uma cópia do Caderno de Tarefas do respectivo nível (Júnior, 1, 2 ou Sênior) para cada competidor.
  • No dia da prova, cada aluno deve ter à sua disposição um computador com editores de texto e pelo menos um compilador entre C, C++, Java ou o interpretador Python.
  • Os alunos devem fazer as provas individualmente, e sem acesso à Internet.
  • Ao final da prova, sugerimos copiar as soluções dos alunos para um pen-drive, como cópia de segurança. Copie apenas os arquivos dos programas-fonte dos alunos (ou seja, o programa do aluno como texto, em uma das linguagens da OBI2018). O nome dos arquivos devem obedecer ao descrito no Caderno de Tarefas.
  • Siga as instruções da página Modalidade Programação - Correção, que será disponibilizada após a prova, para submissão das soluções dos alunos. O procedimento é simples, mas será necessário organizar as soluções dos alunos numa estrutura de diretórios específica, descrita nessa página.
  • A submissão dos programas dos alunos deve ser realizada até {{ ""|obi_date_prova_f1p_coord_submete_solucoes }}.

Solicitamos que mantenham os arquivos de soluções dos alunos, para o caso de algum competidor reclamar da correção.

Caso Especial: Provas no ambiente Saci

Alguns (poucos) alunos estudaram programação pelo ambiente Saci, da OBI, que ensina a linguagem de programação Javascript. Para esses alunos, preparamos uma versão especial do Caderno de Tarefas, para que tenham o mesmo ambiente (e para que possam testar suas soluções em Javascript).

Se algum aluno de sua escola solicitar realizar a prova no ambiente Saci, siga os passos:

  • Faça o download da prova no nível desejado, nos links abaixo (a prova é um arquivo .zip)
  • Copie o arquivo .zip para o computador onde o aluno vai fazer a prova (por exemplo, saci_p1.zip).
  • Descompacte o arquivo. Vai ser criada uma pasta (por exemplo, saci_p1)
  • Dentro da pasta criada, clique duas vezes no arquivo de nome "index.html"; o navegador padrão (Firefox, Chrome, Opera...) do sistema deve abrir e mostrar a prova.
  • Se não funcionar, abra uma janela de um navegador (Firefox, Chrome...) e abra o arquivo "index.html" pelo navegador. A prova deve aparecer como uma página do navegador.
