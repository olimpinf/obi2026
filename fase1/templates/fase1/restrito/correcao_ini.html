Correção - Fase 1 - Modalidade Iniciação

É necessário que os coordenadores realizem a correção das Folhas de Respostas dos competidores. As Folhas de Respostas podem ser corrigidas de duas maneiras:

  • Manualmente, comparando as respostas do competidor na Folha de Respostas com o Gabarito. Cada resposta correta equivale a um ponto.
  • Usando o sistema automático de correção; nesse caso é necessário digitalizar as Folhas de Respostas dos alunos, mas a inserção da pontuação é automática.

O procedimento para correção utilizando cada uma dessas maneiras é detalhado nas seções seguintes.

Correção Manual

Passo-a-passo para Correção Manual

  1. Para cada competidor, compare a Folha de Respostas com o Gabarito do nível correspondente. Os Gabaritos encontram-se disponíveis nos links abaixo.
  2. Contabilize cada questão correta como um ponto.
  3. Insira a pontuação dos competidores, utilizando um dos formulários disponíveis:
    • Para inserir individualmente a pontuação de cada competidor, utilize o formulário
      Insere/Modifica Pontuação
    • Para inserir a pontuação de um lote de competidores, através de um arquivo, utilize o formulário
      Insere Pontuação por Lote

Antes de utilizar, por favor leia as instruções abaixo.

Passo-a-passo para Correção Automatizada

  1. Certifique-se de que o número de inscrição está corretamente preenchido no caso de Folhas de Respostas avulsas (nas Folhas de Respostas pré-prenchidas o número de inscrição já vem corretamente preenchido!).
  2. Não submeta Folhas de Respostas em branco, de competidores que não compareceram.
  3. Separe as Folhas de Respostas dos competidores em lotes, separados pelo nível (Júnior, 1 e 2).
  4. Para cada lote (contendo Folhas de Respostas de um único nível), digitalize (escaneie) as Folhas de Respostas, e salve em um arquivo de formato PDF. Muitos aplicativos permitem salvar várias páginas digitalizadas em um único arquivo, geralmente de tipo PDF. Ou seja, o aplicativo gera um único arquivo no formato PDF contendo as imagens digitalizadas.
  5. Para cada arquivo de Folhas de Respostas digitalizadas, utilize o formulário Correção Automatizada para submeter para a correção automatizada.
  6. O resultado da correção será enviado para o endereço de correio eletrônico do Coordenador Local da OBI.
  7. A pontuação dos competidores será inserida automaticamente, mas solicitamos que os Coordenadores verifiquem nos arquivos recebidos por email se está tudo correto, se as Folhas de Resposta foram corretamente corrigidas. Os arquivos indicam condições de erro como Número de Inscrição incorreto ou mal preenchido, dificuldade de processar o arquivo submetido, etc.)
  8. Verifique a pontuação inserida automaticamente, utilizando o formulário Insere/Modifica Pontuação para consultar as pontuações dos competidores.
  9. Em caso de erros na correção automatizada de um competidor, siga o procedimento para correção manual para esse competidor.

A correção automatizada é um serviço oferecido gratuitamente para a OBI pelo site Web ScanGrader (https://web.scangrader.com/).
