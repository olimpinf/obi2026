Instruções para uso do Corretor

  • Clique apenas uma vez o botão Enviar. O envio do arquivo pela Internet pode demorar (até alguns minutos), principalmente para arquivos grandes, com muitas provas.
  • O resultado da correção será enviado para o endereço de email informado.
  • No campo Referência deve ser informado um identificador para o lote de provas, que será usado para rotular a mensagem de email.
  • O Arquivo de Folhas de Respostas deve ser um arquivo no formato pdf contendo as provas escaneadas (preferência tons de cinza, densidade 150 dpi).
  • O Arquivo de gabarito deve ser um arquivo texto contendo uma linha para cada questão da prova. A linha deve conter o número da questão, seguido por ponto, seguido por espaço, seguido por uma letra (correspondente à alternativa correta). Se a letra for substituída por um asterisco ('*'), qualquer resposta marcada é considerada correta. Se a letra for substituída por um sinal de menos ('-'), nenhuma resposta é considerada correta (significa que a questão foi anulada). Você pode ainda substituir a letra por duas letras separadas por uma vírgula, e nesse caso duas respostas são consideradas corretas. Uma linha iniciando com o caractere '#' é ignorada e pode ser usada para incluir um comentário no gabarito. Um exemplo de gabarito para um teste de 10 questões é mostrado abaixo: 
    # Gabarito para a Prova Exemplo
1. A
2. B
3. A
4. D
5. *
6. C
7. B
8. -
9. B,E
10. A

    No exemplo, a primeira linha é um comentário e é ignorada. Para a Questão 5 qualquer alternativa é considerada correta; para a Questão 8, nenhuma alternativa é considerada correta; e para a Questão 9, as alternativas B e E são consideradas corretas. {% endblock browse %} {% endblock browse-wrapper %} {% block breadcrumbs-wrapper %}
