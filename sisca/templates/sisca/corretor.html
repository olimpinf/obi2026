Instruções para uso do Corretor

  • Clique apenas uma vez o botão Enviar. O envio do arquivo pela Internet pode demorar (até alguns minutos), principalmente para arquivos grandes, com muitas folhas de respostas.
  • O resultado da correção será enviado para o endereço de email informado.
  • No campo Referência deve ser informado um identificador para o lote de folhas de respostas, que será usado para rotular a mensagem de email.
  • O Arquivo de Folhas de Respostas deve ser um arquivo no formato pdf contendo as folhas de respostas escaneadas (preferência tons de cinza, densidade 150 dpi). Não envie arquivos com mais do que cerca de 100 folhas de respostas para correção; se necessário divida as folhas de respostas em vários arquivos.
  • O Arquivo de gabarito deve ser um arquivo texto contendo uma linha para cada questão da prova. A linha deve conter o número da questão, seguido por ponto, seguido por espaço, seguido por uma letra (correspondente à alternativa correta). Se a letra for substituída por um asterisco ('*'), qualquer resposta marcada é considerada correta. Se a letra for substituída por um sinal de menos ('-'), nenhuma resposta é considerada correta (significa que a questão foi anulada). Se a letra for substituída por um sinal de igual ('='), nenhuma resposta é considerada correta (significa que a questão não existe na prova; note que ainda que a questão não seja utilizada na prova é necessário incluir essa informação no gabarito). Você pode ainda substituir a letra por uma sequência letras separadas por uma vírgula, e nesse caso todas as respostas correspondentes às letras da sequência são consideradas corretas. Uma linha iniciando com o caractere '#' é ignorada e pode ser usada para incluir um comentário no gabarito. Um exemplo de gabarito para um teste de 10 questões é mostrado abaixo: 
    # Gabarito para a Prova Exemplo
1. A
2. B
3. A,D,E
4. D
5. *
6. C
7. -
8. D
9. =
10. =

    No exemplo, a primeira linha é um comentário e é ignorada. Para a Questão 3, as alternativas A, D e E são consideradas corretas; para a Questão 5 qualquer alternativa é considerada correta; para a Questão 7, nenhuma alternativa é considerada correta (questão anulada); para as questões 9 e 10 nenhuma alternativa é considerada correta (questões não existentes na prova). {% endblock browse %} {% endblock browse-wrapper %} {% block breadcrumbs-wrapper %}
