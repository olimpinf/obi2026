Para submeter suas soluções em Java, você deve seguir as seguintes instruções.

  • Na prova, cada tarefa tem um título (por exemplo, "Jogo de Dominó") e um nome (por exemplo, "jogo"); o nome da tarefa tem sempre todas as letras minúsculas e sem acento. O arquivo contendo a solução deve ser nomeado nome.java, onde nome é o nome da tarefa. Por exemplo, para a tarefa de título "Jogo de Dominó" e nome "jogo" o arquivo deve ser nomeado jogo.java; para a tarefa de título "Passeios de Dona Minhoca" e nome "minhoca", o arquivo deve ser nomeado minhoca.java.
  • O nome da classe principal pública de sua solução também deve ser o nome da tarefa (em letras minúsculas, sem acento). Por exemplo, para uma tarefa de nome "jogo" a classe principal deve ser declarada assim: 
    import java.util.Scanner;

public class jogo {

    public static void main(String[] args) {
        // seu programa aqui
        ...
    }
}
  • Não use a diretiva package.
  • A entrada de dados é feita pela entrada padrão, a saída deve ser feita na saída padrão.

Um pequeno exemplo

O exemplo abaixo lê dois valores inteiros e imprime o maior valor, para uma tarefa de título "Jogo de Dominó" e nome "jogo" O arquivo deve ter o nome jogo.java. 

import java.util.Scanner;

public class jogo {

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        // lê duas variáveis inteiras
        int a = in.nextInt();
        int b = in.nextInt();

        // escreve o resultado
	if (a > b)
	    System.out.printf("%d\n", a);
	else
	    System.out.printf("%d\n", b);
    }
}
