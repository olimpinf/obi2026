Para submeter suas soluções em Python3, você deve seguir as seguintes instruções:

  • A entrada de dados é feita pela entrada padrão, a saída deve ser feita na saída padrão.

Um pequeno exemplo

O exemplo abaixo lê dois valores inteiros e imprime o maior valor. 

# exemplo de solucao em Python3

# le dois valores inteiros
val1, val2 = [int(i) for i in input().split()]

# escreve o resultado
if val1 > val2:
    print(val1)
else:
    print(val2)

Editor Blockly/Python

Se você aprendeu a programar pelo curso da OBI, você pode desenvolver seus programas usando o Editor Blockly/Python. Note que para acessar o Editor é necessário estar autenticado no site da OBI2022 (como coordenador local, competidor ou usuário saci).

Para submeter seu programa, é necessário copiar o programa Python gerado pelo editor para um arquivo separado e submeter esse arquivo. O arquivo pode ter qualquer nome, mas a extensão deve ser ".py" (por exemplo "solucao.py").
