Para submeter suas soluções em Python2, você deve seguir as seguintes instruções:

  • A entrada de dados é feita pela entrada padrão, a saída deve ser feita na saída padrão.

Um pequeno exemplo

O exemplo abaixo lê dois valores inteiros e imprime o maior valor. 

# exemplo de solucao em Python2

# le dois valores inteiros
val1, val2 = [int(i) for i in raw_input().split()]

# escreve o resultado
if val1 > val2:
    print val1
else:
    print val2
