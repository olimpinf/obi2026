Para submeter suas soluções em Pascal, você deve seguir as seguintes instruções:

  • A entrada de dados é feita pela entrada padrão, a saída deve ser feita na saída padrão.

Um pequeno exemplo

O exemplo abaixo lê dois valores inteiros e imprime o maior valor. 

// exemplo de programa em Pascal

var val1, val2: integer;

begin
  // lê dois valores inteiros
  readln(val1, val2);

  if (val1 > val2) then
    writeln(val1)
  else
     writeln(val2);
end.
