Passo-a-passo para configuração do aplicativo OBS Studio para o ambiente de prova da APIO (Asia-Pacific Informatics Olympiad).

1. Faça o download do aplicativo para o sistema operacional de seu computador

https://obsproject.com

2. Siga as instruções de instalação

Ação depende de cada sistema operacional.

3. Inicie a execução do OBS Studio

Ação depende de cada sistema operacional.

4. Clique no botão Settings da aba "Controls" (canto baixo direito) da janela do OBS

Uma janela de configuração aparece.

5. Ajuste os parâmetros de vídeo

  • Clique no botão Video no menu à esquerda.
  • Escolha o parâmetro Integer FPS Value e coloque o valor 1.
  • Clique em OK para confirmar.

6. Ajuste os parâmetros de áudio

  • Clique no botão Audio no menu à esquerda.
  • Verifique que todos os parâmetros da seção Global Audio Devices estão desabilitados (disabled). Normalmente o parâmetro Mic/Auxiliary Audio inicialmente tem o valor "Default", altere para "Disabled".
  • Clique em OK para confirmar.

6. Ajuste os parâmetros de saída

  • Clique no botão Output no menu à esquerda.
  • Na seção Streaming, altere o parâmetro Video Bitrate para o valor "200 Kbps".
  • Na seção Recording, se necessário altere o parâmetro Recording Path para um diretório adequado.
  • Na seção Recording, altere o parâmetro Recording Quality para "Same as stream".
  • Clique em OK para confirmar.

7. Para iniciar a gravação (imediatamente antes do início da prova)

  • Clique no botão Start Recording da aba "Controls" (canto baixo direito).
  • Clique em OK para confirmar.

8. Para terminar a gravação (imediatamente após o início da prova)

  • Clique no botão Stop Recording da aba "Controls" (canto baixo direito).
  • Clique em OK para confirmar.
