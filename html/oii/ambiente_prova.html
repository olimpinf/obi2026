Ambiente de prova para a OII2023 (Olimpíada Iberoamericana de Informática)

Computador

Se você vai fazer a prova utilizando um computador da escola, não é necessário seguir as instruções sobre o computador.

No computador que será utilizado para a prova deve ser instalado o software OBS Studio. Esse software é gratuito e tem versões para Linux, Windows e MacOS. O OBS Studio será utilizado para gravar a tela do computador durante a competição. Ao final de cada prova, o competidor deverá enviar o arquivo gravado pelo OBS Studio para a Coodenação da OBI.

Veja como configurar o OBS Studio para o ambiente de prova da OII.

Celular

Se você vai fazer a prova na escola, não é necessário seguir as instruções sobre o celular.

Além do computador, o ambiente de prova exigirá também que o competidor utilize um telefone celular conectado a uma sala do Google Meet da Coordenação da OBI. Endereço da sala do Google Meet:

meet.google.com/tii-ayjs-uhi

OmegaUp

Para submissão e correção das soluções será utilizado o sistema OmegaUp (https://www.omegaup.com).

Condições Gerais

Para a OII continuam valendo as condições gerais de provas da OBI: as provas são individuais e sem consulta a qualquer tipo de material (impresso, arquivos digitais, etc.).
