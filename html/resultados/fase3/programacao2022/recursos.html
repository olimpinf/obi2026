Para solicitar recorreção, contate o Coordenador Local de sua escola e juntos sigam o seguinte roteiro:

  • Baixem os gabaritos (conjuntos de testes) para a tarefa que deseja solicitar a correção. Os gabaritos contém os arquivos de entrada e saída usados na correção.
  • Compilem e testem o programa com cada arquivo de entrada, verificando que o programa produz o mesmo resultado que o arquivo de saída correspondente.
  • Se (e somente se) o programa produz o resultado esperado para os gabaritos, o Coordenador Local deve enviar uma mensagem para

informando:

  • o número de inscrição do competidor
  • o nome do competidor
  • o nome da tarefa para a qual deseja solicitar recorreção
  • a razão da solicitação (compilação, execução, biblioteca padrão faltante, etc.)

Solicitamos a todos os Coordenadores que realmente certifiquem-se, executando localmente o programa do aluno com todos os casos de testes disponibilizados, de que é o caso de solicitar um recurso. Os recursos são processados manualmente e dão trabalho!

Serão considerados recursos enviados até 21:00 horas de terça-feira, dia 25 de outubro de 2021.
