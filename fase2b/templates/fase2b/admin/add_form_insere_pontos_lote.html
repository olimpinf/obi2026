Instruções

Formato do arquivo

O arquivo deve ter o formato padrão CSV (do inglês "Comma Separated Values", ou seja, "Valores Separados por Vírgula"). Aplicativos de planilhas, como OpenOffice e Excel, permitem salvar o arquivo nesse formato.

A planilha deve ter os seguintes campos, na ordem dada:

  • Num. Inscr.: número de inscrição.
  • Nível: deve ser IJ (iniciação júnior), I1 (iniciação 1) ou I2 (iniciação 2).
  • Pontos: Pontos (cada questão certa vale um ponto)

Passo a passo

  1. Obtenha a planilha vazia, já no formato correto, clicando aqui.
  2. Edite a planilha, inserindo os dados dos competidores.
  3. Salve a planilha em um arquivo no formato CSV.
  4. Envie o arquivo utilizando o formulário acima.
