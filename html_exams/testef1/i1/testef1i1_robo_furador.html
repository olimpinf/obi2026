Um robô furador pode ser programado usando os comandos gráficos → e ◎, que realizam as seguintes operações :

  • → : robô move-se 1cm para a frente
  • ◎ : robô faz um furo na posição corrente

Assim, podemos programar que o robô faça dois furos a uma distância de 1cm um do outro com os comandos ◎ → ◎. Além disso, podemos programar repetições, utilizando números e parênteses. Por exemplo

  • 4 → : repete quatro vezes a ação "move-se 1cm para a frente" (ou seja, o robô move-se 4cm para a frente)
  • 3 ◎ : repete três vezes a ação "faz um furo na posição corrente" (como o robô não se move, faz um único furo na posição corrente)
  • 2 ( → → ) : repete duas vezes a ação "move-se 1cm para a frente, move-se 1cm para a frente" (ou seja, o robô move-se 4cm para a frente)
