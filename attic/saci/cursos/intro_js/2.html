Comando de escrita: printf

O comando printf é usado para escrever os resultados do programa. Na sua forma mais simples, o comando printf escreve uma cadeia de caracteres passada como argumento entre aspas duplas. O comando deve sempre terminar com um ponto e vírgula.

Exemplo

printf("Meu primeiro programa\n");

Os caracteres \n (barra invertida seguida da letra 'n' minúscula) na cadeia de caracteres passada como argumento fazem com que uma quebra de linha seja escrita na saída.

Outros Exemplos

printf("Estava à toa na vida\n");uma linha é escrita, com uma quebra de linha após 'vida'
printf("O meu amor me chamou\n");uma linha é escrita, com uma quebra de linha após 'chamou'
printf("Prá ver\n a banda passar\n");duas linhas são escritas, com quebras de linha após 'ver' e 'passar'
printf("O tempo e o vento\n\n\n");três linhas são escritas, com quebras de linha seguidas após 'vento'
printf("2 mais dois é igual a 4\n");uma linha linha é escrita, com quebra de linha após '4'

Comentários

Comentários são usados para documentar o programa. Eles são colocados no programa para passar informações para quem vier a ler o programa (como colaboradores, ou o próprio autor). São usados por exemplo para explicar o algoritmo utilizado, ou informar o autor do programa, ou qualquer outra informação interessante. Eles são desconsiderados na execução do programa.

Em Javascript, comentários são iniciados com duas barras (//), e se estendem até o final da linha.

Exemplos

// Autor: Ricardo Anido

// Este trecho de programa calcula o resultado final

// printf("OK\n"); note que este comando não é executado, é um comentário!

Praticando o comando printf

Escreva um programa que produza a saída mostrada na seção Saída esperada, abaixo. Note que você pode escrever o programa de muitas maneiras! Escreva duas versões:

  • uma que utiliza dois comandos printf, cada comando produzindo uma linha (não se esqueça que cada linha deve ser terminada com o caractere final de linha "\n");
  • uma que utiliza apenas um comando printf (especificando os finais de linha com "\n" na cadeia formatadora).

Note ainda que, exceto pelos caracteres final-de-linha ao final de cada linha, não há espaços em branco na saída mostrada abaixo. Portanto seu programa não deve produzir espaços em branco adicionais. A saída deve ser exatamente como mostrado abaixo.

Saída esperada

Oi!
2+2=4
